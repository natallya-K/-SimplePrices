INSTALLER LA SOLUTION DANS VISUAL STUDIO
1. créez un clone dans fork
2. lancez Visual Studio en avec les droits d'admininstrateur
3. ouvrez la solution SimplePrices.sln
4. en bas à droite, faire un nouveau Fetch de dev
5. via Générer / regénérez la solution (ce qui télécharge les fichiers manquants de visual studio)
7  activier toutes les dépendances : Générer / Régénerer la solution
6. démarrez la solution pour voir si le serveur se lance

INSTALLER LA BASE DE DONNEES DANS SQL SERVER EXPRESS
1. démarrez le MS management studio avec les droits d'administrateur
2. créez une nouvelle base de données "simpleprices"
3. ouvrez la requète SQL à partir de d:\simpleprices\DB\
4. lancez la requète
